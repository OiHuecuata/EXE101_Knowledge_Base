# EXE101_Knowledge_Base
[English description below]

Giảng viên của tôi trong buổi học đầu tiên của môn EXE101 đã từng kể về chuyện nhét 1000 câu hỏi vào đầu chỉ trong vòng nửa ngày. Tôi thấy mình nhét không nổi nên đành nhét tạm vào LLM.

When my EXE101 professor shared the challenge of mastering 1,000 questions in half a day, I decided to engineer a better solution than just rote memorization. I built an AI Agent to 'absorb' it for me. This project is an end-to-end RAG system that turns months of course content into an instantly searchable and intelligent academic companion.
